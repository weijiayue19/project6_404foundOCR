01 / OVERVIEW

产品概览

先花一分钟认识404 Found OCR：能读什么、数据放在哪儿，以及两种识别模式该怎么选。

SUPPORTED INPUT 图片 + PDF JPG / JPEG / PNG / BMP / PDF
RECOGNITION MODE 双模式 快速识别 / 深度识别
DATA LOCATION 本地保存 SQLite 历史记录与本机推理
BATCH PROCESS FIFO 队列 多图按当前缩略图顺序识别

一份文件会依次经过导入、检查预览、可选编辑或预处理、PaddleOCR 识别，最后来到结果编辑、 SQLite 历史保存和导出。流程不绕，跟着界面走就好。
普通截图和清晰图片先用快速识别；遇到 PDF、复杂排版、标题段落或表格，再把任务交给深度识别。

离线说明： OCR 推理、历史存储和查询都在本机完成。只有模型还没下载时，第一次启动可能需要联网获取 官方模型文件；下载完成后，就可以安心在本地使用了。
01 默认推荐

快速识别

  • 适合截图、课堂资料和普通印刷体图片。
  • 使用轻量 OCR 服务，在 CPU 上通常等得更短。
  • 可执行整图识别、选区识别和批量图片识别。
  • 导入 PDF 时，系统会自动切换到深度识别。
02 复杂文档

深度识别

  • 适合 PDF、标题、段落、复杂版面和表格文档。
  • 使用 PP-StructureV3，看得更细，也需要多一点耐心。
  • PDF 会直接进入文档识别流程，不执行图片选区和手写预处理。
  • CPU 推理可能会久一些，看看流程状态条，小恐龙会替你报进度。

macOS 首次打开提示

当前 macOS 版本尚未完成 Apple Developer ID 签名与公证。首次打开时，macOS 可能提示“无法验证开发者”或“Apple 无法检查其中是否包含恶意软件”。请仅在确认安装包来自本项目官方发布渠道时继续打开。

方法一：右键打开
  1. 将应用拖入“应用程序”文件夹。
  2. 在“访达”→“应用程序”中找到 404 Found OCR。
  3. 按住 Control 键点按应用图标，或右键点按应用图标。
  4. 在菜单中选择“打开”，然后再次点击“打开”。
方法二：在系统设置中放行
  1. 双击应用，等待 macOS 显示安全提示，然后关闭提示窗口。
  2. 打开“系统设置”→“隐私与安全性”。
  3. 向下滚动至“安全性”区域，找到有关 404 Found OCR 被阻止的提示。
  4. 点击“仍要打开”，再次确认“打开”，并按系统提示输入登录密码或进行身份验证。

如果系统提示“应用已损坏”或“将损害你的电脑”，请不要继续放行，应重新下载安装包或联系开发者。

Windows 安装包可能遇到的问题

Windows 安装包因 SmartScreen、Smart App Control 或缺少运行库等原因，可能出现安装或启动拦截。请按照下列建议处理，不要绕过系统安全功能，只有确认安装包来源可信时再继续操作。

方法一：出现“Windows 已保护你的电脑”时
  1. 确认安装包来自本项目官方发布渠道。
  2. 在提示窗口中点击“更多信息”。
  3. 核对应用名称和发布者信息后，点击“仍要运行”。
  4. 如果没有“仍要运行”选项，请不要关闭系统安全功能，建议重新下载安装包或联系开发者。
方法二：安装包属性中显示“解除锁定”时
  1. 在文件资源管理器中找到下载的安装包文件。
  2. 右键单击安装包，选择“属性”。
  3. 在“常规”选项卡底部找到“安全”区域。
  4. 如果显示“解除锁定”，勾选该选项。
  5. 点击“应用”→“确定”，然后重新运行安装包。

此操作仅适用于下载文件被 Windows 标记为来自网络来源的情况。若系统仍因 Smart App Control 阻止未签名应用，则需要使用签名安装包或联系开发者，不应关闭整个安全功能。

03 / WORKFLOW

从图片到文字的完整流程

不用记住复杂步骤，照着真实操作顺序就可以轻松理解。

启动程序并选择识别模式

在启动页点击“快速识别”或“深度识别”；也可不进入工作台，直接打开“历史查询”。

准备好，进入对应的 OCR 工作台

导入图片或 PDF

怎么顺手怎么来：点击“添加图片/添加文档”、把文件直接拖进窗口，或复制图片后按 Ctrl/Command + V。

文件会排进预览区和缩略图队列

检查并编辑当前图片

通过上一张、下一张和缩略图切换文件；必要时旋转 90°、水平镜像或上下镜像。

放心调整，原文件不会被改动

选择识别范围与预处理

整图识别无需框选；局部文字可在预览区拖拽框选。低质量图片可打开预处理并自由调整处理顺序。

少一点背景干扰，多一点识别把握

开始识别

点击“识别整图/识别文档”，或对已框选区域点击“识别选区”；图片载入后也可按Enter直接开始。

状态条会显示：等待、处理中或已完成

核对、编辑与导出结果

在右侧文本区直接修改内容，可复制、导出TXT、保存编辑文本或切换排版视图。

文字到手，可以继续编辑和使用

历史查询与记录管理

输入关键词，组合文件类型、识别模式和日期范围筛选，查看缩略图、完整文本或删除记录。

不用重新 OCR，也能找回以前的内容

01

快速识别卡片中的“开始使用”

界面位置：启动页右上方第一张模式卡片

作用：进入普通图片快速 OCR 工作台。

用户怎么用：识别截图、课件、笔记和清晰印刷体图片时优先选择——日常小任务交给它，通常更省时间。

执行结果：工作台顶部显示“快速识别”，使用轻量 OCR 服务。

02

深度识别卡片中的“开始使用”

界面位置：启动页右侧第二张模式卡片

作用：进入 PDF 与复杂文档识别工作台。

用户怎么用：遇到标题、段落、复杂版面、表格或 PDF 时选择；文档越“有层次”，越适合让它上场。

执行结果：使用 PP-StructureV3，CPU 推理时间可能明显增加。

03

历史查询

界面位置：启动页右下角

作用：无需进入 OCR 工作台，直接查看本地已保存记录。

用户怎么用：已经识别过内容、只想把文字找回来时直接点击，不必让 OCR 再跑一圈。

执行结果：打开独立历史查询窗口，不会重新调用 OCR。

04

设置

界面位置：启动页右上角齿轮按钮

作用：打开独立的应用设置窗口，调整本地 OCR 的辅助功能。

用户怎么用：点击齿轮后，可启用或关闭“小恐龙桌面助手”；需要帮助时，点右下角的“？”就能回到这本手册。

执行结果：关闭桌面助手时，当前显示的小恐龙桌面宠物会隐藏；设置窗口不会启动 OCR 或改变识别模式。

B. OCR 主工作台

工作台把各项任务分了区：顶部负责发号施令，左侧照看图片，右侧接收文字，中间的小恐龙状态条负责报告进度。

05

返回

界面位置：工作台左上角，PIXEL VISION 左侧

作用：返回启动页，重新选择快速或深度识别。

使用提醒：手动改过文字时，记得先保存再返回，别让认真校对的成果悄悄溜走。

06

识别模式

界面位置：工作台右上角

作用：在快速识别与深度识别之间切换。

使用方式：识别任务未运行时点击；任务进行中不可切换。

注意：两种模式的结果分别缓存和保存，不会互相覆盖；从深度识别切换到快速识别时，任务列表中的 PDF 文档会被移除，图片仍会保留。

07

添加图片 / 添加文档

界面位置：主工具栏最左侧

作用：选择一个或多个本地文件，追加到当前队列。

使用方式：快速模式选择图片；深度模式可选择图片和 PDF。

结果：文件进入预览和缩略图队列，但不会自动开始识别。

08

识别选区

界面位置：主工具栏，“添加图片”右侧

作用：单图时只识别当前图片框选的局部区域；多图时按队列批量处理各图片。

使用方式：先在左侧预览图按住鼠标左键拖拽，再点击该按钮。多图队列中，有选区的图片识别选区，未框选的图片自动识别整图。

适合：页面中只有一小块文字有用，或背景干扰较多。

09

清除选区

界面位置：主工具栏，“识别选区”右侧

作用：取消当前预览图片已保存的框选区域。

使用方式：切换到需要处理的图片后点击；该操作不会清除队列中其他图片的选区。

结果：之后可重新框选，或直接使用整图识别。

10

识别整图 / 识别文档

界面位置：主工具栏黑色主要按钮

作用：启动当前图片、当前队列或 PDF 文档的 OCR 任务。

使用方式：检查顺序、方向、选区和预处理后点击，也可按 Enter。

结果：状态条开始运动，完成后右侧显示文本并写入历史库。

11

预处理

界面位置：主工具栏右侧辅助操作区

作用：配置灰度化、自适应二值化和中值滤波。

使用方式：勾选必要步骤，并拖拽改变执行顺序。

注意：清晰图片通常不需要把选项全部打开；预处理不是越多越好，给笔画“加工过头”反而可能帮倒忙。

12

历史查询

界面位置：主工具栏右侧，“预处理”旁

作用：从工作台打开独立的历史记录窗口。

使用方式：查找之前识别过的图片、PDF 和编辑文本时点击。

09 / LIMITS

当前版本使用边界

正式开工前可以先扫一眼这里。哪些文件能处理、哪里需要多等一会儿、哪些能力还在路上，都写清楚了。

输入限制

支持 JPG、JPEG、PNG、BMP 图片和 PDF 文档。单个文件不超过 200 MB，单张图片总像素不超过 5 亿。

PDF 限制

PDF 只进入深度识别，不支持图片旋转、镜像、选区裁剪或手写图像预处理。

性能提示

PP-StructureV3 在 CPU 上可能需要多一点时间。批量与深度识别时看着流程状态条即可，小恐龙还在工作，就先别连续敲识别按钮。

结果校对

模糊、倾斜、复杂背景和特殊字体都会影响准确率。重要材料别急着交卷，人工核对后再保存或导出更稳妥。

当前导出

目前可以复制文字或导出 TXT。JSON、标注图片和结构化表格富文本还在后续计划里，暂时先别在菜单里找它们。

任务控制

批量任务会逐张回传进度；任务取消、每张图片的详细耗时和单图线程取消还在继续完善。

10 / FAQ

常见问题

导入 PDF 后为什么自动变成深度识别？

PDF 由 PP-StructureV3 文档服务处理，所以快速模式收到 PDF 后会自动切到深度识别。模式切好以后，再点击“识别文档”即可。

为什么导入文件后没有立即开始识别？

我们把开始键留给你，是为了让你先检查文件、调整顺序、旋转镜像，或设置选区和预处理。准备妥当再识别，返工会少很多。

预处理应该全部打开吗？

不用。清晰图片直接识别通常就很好；光照不均时可试试自适应二值化，椒盐噪声较多时再用中值滤波。预处理开得太满，反而可能伤到笔画。

为什么深度识别等待时间较长？

深度模式要分析更完整的文档结构，在 CPU 上自然会比快速模式多花一些时间。等候时可以看看状态条，也可以打开小恐龙窗跑两圈。

修改后的文字会覆盖模型原文吗？

不会。点击“保存编辑文本”后，模型原文会好好留在数据库里，你修改后的版本也会单独保存，之后可以在历史详情中来回查看。

历史查询会重新消耗 OCR 时间吗？

不会。历史查询只是读取本地 SQLite 中已经保存的记录，不会再叫 PaddleOCR 重做一遍。
